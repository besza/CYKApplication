CYK-algoritmus implementáció Spark Java keretrendszerrel

Bevezető

A Cocke-Younger-Kasami algoritmus egy olyan alulról-felfelé haladó szintaktikai elemző, amely egy tetszőleges λ-mentes Chomsky-féle normálformában lévő nyelvtanhoz egy adott input szó esetén megkonstruálja a levezetési fát és végeredményül eldönti, hogy az adott input szó eleme-e a nyelvtan által generált nyelvnek.

Használat

A nyelvtanunk szabályait az alábbi formában adhatjuk meg: egy nemterminális karaktert nyíl követ, majd a lehetséges alternatívák felsorolva | jellel tagolva. Egy lehetséges szabály tehát így nézhet ki: S->AB|C|a|b Az alkalmazás feltételezi, hogy a kezdőszimbólum minden esetben S. Szükséges továbbá, hogy a nyelvtanunk Chomsky-féle normálformában legyen, ez azt jelenti, hogy minden szabály kizárólag ilyen alakú lehet: A->BC vagy A->a. Konvenció szerint minden nagybetű nemterminális, illetve minden kisbetű terminális szimbólumot jelöl.
